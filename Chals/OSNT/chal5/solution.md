# OSINT Challenge - Who is behind @flyinhigh5_?

## Challenge
Linked to the handle **“@flyinhigh5_”**, a name and email are yours to find.  
Who’s the person behind this trace?

## Step-by-Step Solution

1. **Start with the handle:**
   - The Twitter/X handle provided is `@flyinhigh5_`.

2. **Profile Analysis:**
   - Searching the handle shows connections to **Liverpool FC fandom**.
   - From contextual clues, we can link the persona to the name **Yahya Rehman**.

3. **Cross-Platform Investigation:**
   - Pivoting from the username and interests, we check other platforms like **GitHub**.

4. **GitHub Profile Found:**
   - A GitHub account exists with the username [yahyalfc](https://github.com/yahyalfc).
   - The GitHub profile displays:
     - **Name:** Yahya Rehman  
     - **Email:** yahyarehmanlfc@gmail.com  


## What You Learned
- Always **pivot usernames** across platforms (Twitter → GitHub in this case).  
- Public GitHub profiles may expose **real names** and even **emails** if the user has them linked.  
- OSINT investigations often involve linking **social media presence with technical profiles**.
