Pwn Lab — Challenge 1

Buffer Flip

A tiny program asks for your name. The developer made a mistake and allowed too many bytes to be read into a small buffer. Find the bug and make the program print the flag (flag.txt is present in the challenge folder).

What to submit: The flag printed after you trigger the bug.

Hints:
  • Look at the sizes of variables in the source code.
  • Try sending more characters than the buffer holds.
  • Use only safe local tools (run in a lab VM).

Files expected in this folder: chal1 (binary), challenge.c
